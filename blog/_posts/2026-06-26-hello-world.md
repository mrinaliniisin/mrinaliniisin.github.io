This is the first post on my new blog. It's written in **Markdown** and
rendered by Jekyll, which GitHub Pages builds automatically every time I push.

## Writing a new post

To add a post, drop a Markdown file into `blog/_posts/` named like:

```
YYYY-MM-DD-some-title.md
```

Start it with a front-matter block:

```yaml
---
layout: post
title: "Your title here"
date: 2026-06-26
---
```

…then write below it. Headings, **bold**, _italics_, [links](https://example.com),
lists, and `code` all work:

- It shows up automatically on the [blog index](/blog/).
- The URL becomes `/blog/some-title/`.
- No HTML required.

That's the whole workflow. Happy writing!
